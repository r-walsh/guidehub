##GuideHub

This site is being constructed to serve as a hub for curating, storing, and tracking guides on various topics, with a focus on programming.

###Technologies used:
- Frontend:
    + React
    + ES6
    + Flux
    + LESS
    + Webpack
- Backend:
    + Node/Express
    + ES6
    + Passport/Passport-GitHub
    + Babel
    + MongoDB
